# Social Network Service

* Facebook을 모티브로한 SNS 프로젝트

## 사용기술

Spring Boot, Maven, MyBatis, MySQL, Java, IntelliJ

## 프로젝트 관리 

* Wiki UserCase 작성
* UserCase 이슈로 등록 후 각각 브랜치 생성
* git-flow를 이용한 브랜치 전략

## 프로젝트 예상 이미지
